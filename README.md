project Name: Fitness Track App
Description
This is a full-stack web application that allows users to build and complete personalized workout tracks. Users can select exercise categories, specify the duration of their workout, perform the exercises, and give themselves a score for each exercise. At the end of the workout, a summary is displayed and the results are saved to the database.

Technologies Used
React – for the frontend

Redux Toolkit – for state management

ASP.NET Core (C#) – for the backend API

SQL Server – for storing data

Main Features
Create a custom workout by selecting categories (e.g., Abs, Legs)

Each exercise is displayed with a matching image or animation

Users score themselves after each exercise

A final summary screen shows all scores and results

Users can log in and view past workouts

Smart suggestions based on previous scores and preferences

Project Structure
client/ – the React frontend

components/ – reusable UI parts

pages/ – main application views

server/ – the ASP.NET Core backend

Controllers/ – API endpoints

Models/ – data models like Exercise, Track, etc.

Data/ – database context and access logic

Insert_FitnessTracks_Data.sql – a script to insert sample data into the database
